# Como Contribuir 

O código-fonte do site está disponível [no GitHub](https://github.com/titerman/mapa-especial) sob a licença MIT. Agradecemos contribuições externas para melhorar o site e seu conteúdo.

## Adicione ou edite dados do site

Contribuidores podem criar *pull requests* para adicionar novos dados ou remover dados incorretos. Os dados são armazenados no formato GeoJSON, que é fácil de ler e editar. Crie um fork de esse repositório na sua conta do GitHub, edite os arquivos e abra um pull request. Leia o GitHub README em [inglês](https://github.com/titerman/mapa-especial/blob/main/README_EN.md) ou [português](https://github.com/titerman/mapa-especial/blob/main/README.md) para saber mais.

## Adicione ou melhore a funcionalidade do site

Você é bem-vindo para enviar PRs com melhorias para o site. Os mantenedores têm autoridade para recusar PRs de usuários, então aprove sua ideia com o proprietário do repositório antes de colocar qualquer trabalho. Certifique-se de que seu PR não quebre a funcionalidade existente ou tenha impacto negativo na aparência do site.

## Relatórios de bugs

Crie um novo [ticket do GitHub](/) se notar um bug ou tiver uma ideia de melhoria.
